CPU Register Viewer

This viewer shows all the cpu registers. When in run mode these are view only, but in other modes they can be edited. To edit click on the register you want to change and type the hex value you want to change it to. When you press enter the value will take effect.

Registers

PC: Program Counter

A: Accumulator

X: X register

Y: Y register

S: Stack Pointer

Status Flags

N: Negative

V: Overflow

B: Break

D: Decimal

I: Interrupt

Z: Zero

C: Carry
