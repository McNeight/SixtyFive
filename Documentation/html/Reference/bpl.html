BPL

Branch on Plus

Function:

Branches is the negative flag = 0. The second byte of the branch instruction contains a signed displacement (+127 to -128). The displacement is added to the address of the instruction following the current instruction to get the final address.

Addressing Modes:

  IMP ACC ABS ZP IMM ABS,X ABS,Y IND,X IND,y ZP,x ZP,y REL

IND
