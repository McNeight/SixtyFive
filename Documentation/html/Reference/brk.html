BRK

Break

Function:

This instruction functions similar to an external IRQ interrupt. When executed the program counter + 2 and status register will be pushes onto the stack with the status registers break bit set to 1. To exit a break handler use the RTI instruction. Note that since the program counter + 2 is pushed onto the processor will not return to the instruction after the BRK, but will return to one address later.
