Processor Status Register

Bit 7 6 5 4 3 2 1 0
Flag N V   B D I Z C



N = NEGATIVE. Reflects the state of bit 7 after an operation. Used to indicate negative values.

V = OVERFLOW. The overflow flag is set during arithmetic operations if the result has yielded an invalid 2's complement result (e.g. adding to positive numbers and ending up with a negative result: 64 + 64 => -128). It is determined by looking at the carry between bits 6 and 7 and between bit 7 and the carry flag.

B = BRK COMMAND. Set to 1 if an IRQ is caused by a BRK, reset to 0 if caused by an external IRQ.

D = DECIMAL MODE. Set to 1 to enable decimal math mode..

I = IRQ DISABLE. Set to 1 to disable IRQ interrupt.

Z = ZERO. Set to 1 if the result of the last operation was zero.

C = CARRY. Set if an add produced a value larger then 255, or if the subtraction
produced a borrow. Also used during logical shifts.
