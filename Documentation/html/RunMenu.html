Run Menu

Stop [F5]

Stop the simulation that was started with either the Start or Animate command and returns to idle mode.

Start [F6]

Starts the simulation in run mode. In this mode the viewers will not update until the simulation is stopped by either the stop command or a breakpoint.

Step [F7]

Singles steps the simulation. The next instruction will be executed and the viewers will be updated.

Animate [F8]

Starts animate mode. In this mode the simulator will run at a speed determined by the speed drop down on the tool bar and the viewers will be updated after every instruction. Animate mode will continue until the either the stop command is selected or a breakpoint is reached.

Reset [F12]

Resets the simulated processor. The program counter will be loaded from the reset vector at locations $FFFC/$FFFD.
