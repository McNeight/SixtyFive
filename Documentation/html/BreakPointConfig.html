Break Point Configuration


This dialog box is used to set breakpoints. When the simulator is in run mode and it encounters any break condition execution will stop and the simualtor will return to stop mode.

Address Breaks

An address break will cause execution to stop when the instruction at the break address is read.

Add - Type a hex address in the Address box and click Add to add the address break
Delete - Click on address in the address break box and click Delete to delete it.
Clear - Click clear to clear all address breaks.
Enable - Click this to enable address breaks.

Data Breaks

A data break occurs when a certain value is read from or written to a specific address. If not data is specified from a data break then only the read or write is checked for, not the actual value.

Add - To add a data break put the hex address in the Address box, the data (if any) in the Data box, select Read or Write, and click Add.
Delete - Click on the data break in the list and click Delete to delete it.
Clear - Click this to clear all data breaks.
Enable - Click this to enable data breaks.

Clear All - Click this button to clear all data and address breaks.
