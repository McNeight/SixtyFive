1.0 Introduction

SIM6502.NET is a generic 6502 processor simulator. It simulates a basic 6502 system that can be used as a learning tool, or as an easy way to develop and test 6502 code. This file gives you the basic information needed to install SIM6502.NET. To learn how to use the program open the sim6502.chm help file.

2.0 Installation

2.1 Prerequisites

To run SIM6502.NET, you will need to have the .NET runtime installed onto your computer. You can download it from here:

http://msdn.microsoft.com/netframework/downloads/framework1_1/

2.2 Installation

Unzip the files into a temporary directory and run setup.exe to install the program and it's support files. You can start the program from the Start menu, or by running sim6502.exe from the installation directory.
